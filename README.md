# 🤖 AI-Powered HR Attrition Insights & Decision Assistant

## 📌 Overview
This project demonstrates how AI (via structured prompt engineering) can enhance HR analytics by transforming workforce data into actionable business insights.

Built as a practical use case for HRIS and People Analytics professionals, this project combines:
- HR data analysis
- Dashboard visualization
- AI-generated insights and recommendations

---

## 🎯 Objective
To create an AI-assisted workflow that:
1. Analyzes employee attrition data  
2. Identifies key trends and risk factors  
3. Generates executive-ready insights  
4. Recommends data-driven HR actions  

---

## 🛠️ Tools & Technologies
- Excel / Power BI (data analysis & visualization)
- Python (optional – predictive modeling)
- ChatGPT (AI insights generation)

---

## 📊 Dataset
Sample HR dataset containing:
- Employee ID
- Department
- Tenure
- Salary
- Performance Rating
- Attrition (Yes/No)

(Source: Kaggle HR Employee Attrition Dataset)

---

## 📈 Project Workflow

### Step 1: Data Preparation
- Cleaned and structured employee dataset
- Validated missing and inconsistent values

### Step 2: Dashboard Creation
Built a dashboard to track:
- Attrition rate
- Attrition by department
- Attrition by tenure
- Compensation vs attrition trends

### Step 3: AI-Powered Insights (Prompt Engineering)

Used structured prompts to:
- Identify attrition drivers
- Analyze workforce risks
- Generate HR recommendations
- Create executive summaries

## 📌 Key Insights

- Employees with less than 2 years tenure show the highest attrition risk  
- Sales and Customer Support departments have elevated turnover  
- Younger workforce segments (<30) show higher attrition trends  
- Compensation range $60K–$100K has highest employee concentration  

---

## 💡 Recommendations

- Implement targeted retention programs for early-tenure employees  
- Conduct department-specific engagement initiatives  
- Review compensation alignment for high-risk roles  
- Improve internal mobility and career progression pathways  

---

## 🚀 Business Impact

- Reduced manual HR analysis effort using AI-assisted insights  
- Enabled data-driven decision-making for workforce planning  
- Identified key attrition drivers to improve employee retention strategy
